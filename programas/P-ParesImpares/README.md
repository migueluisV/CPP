## Bienvenidos
Este programa fue realizado en Visual Studio Community 2019, con el paquete de trabajo *Desarrollo para el escritorio con C++*.
## Estructura de la carpeta
Esta carpeta contiene dos subfolders y un archivo, los cuales son:
- `Debug`: contiene el archivo ejecutable en consola del proyecto, además de un fichero *.pdb*.
- `P-ParesImpares`: contiene el código fuente *.cpp*, clases creadas y otros ficheros importantes.
- `P-ParesImpares.sln`: es el fichero que abre Visual Studio para trabajar y compilar el código fuente.
## Descripción del programa
El objetivo de este programa es repasar un poco sobre la sintaxis de C++, el uso de funciones, ingreso y despliegue de valores, utilización de ciclos, etc.
